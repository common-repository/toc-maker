Table of contents Maker automatically creates a table of contents from headings.
== Description ==

Table of contents Maker is a plugin that checks the headings in an article and automatically inserts a table of contents.

== Installation ==

1. Upload `toc-maker` folder into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can change the default settings and more under Settings > Table of contents Maker
